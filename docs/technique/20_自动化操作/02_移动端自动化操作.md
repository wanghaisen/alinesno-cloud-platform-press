# 移动端自动化操作
 
## 概述

这里集成的移动端自动化能力，主要是集成chatops的能力，当前主要是集成钉钉的开发能力，整合起来chatops体系，便于移动化运维和监控

## 本内容你将获得

- 移动端集成的方式
- chatops体系能力
- 安全性的处理和使用场景
- 相关注意事项

## 集成架构设计

<img src="/operation/chatops_01.jpg" style="width:80%">

架构说明：

- 集成dingtalk等监控服务，发送指令到自动化服务
- 自动化服务通过自定义的脚本，操作多个服务平台
- 执行结果返回会话(Dingtalk)
- 结果返回各个结果，可自定义发送到邮件、dingtalk、短信等

## 集成ChatOps能力

主要是针对于快速性的打造自动化的能力，去掉重复性的工作

12. 自动安装mysql数据库
13. 自动添加mysql用户

15. 自动添加jenkins用户账号
16. 自动操作阿里云

18. 自动查询运维监控情况

21. ...

### 集成方式

这里主要说明钉钉的集成方式

- 获取到AccessToken和机器人
- 配置请求链接和配置通知机制
- 集成验证

### 注意事项

服务当前需要安装的内网，同时注意网络策略和安全的问题。
