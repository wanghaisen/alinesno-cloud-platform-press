# 访问策略配置

## 概述

访问策略页用来配置路由的生效时间，超时时间，ip限定等等

## 本内容你将获得

- 路由的策略配置

## 路由的策略配置
在网关管理平台中导航网关管理>访问策略，此为路由策略配置页，在此页配置了策略项后可以在
<img src="/technique/gateway/08_gateway_policy.png" width="70%">

配置说明:

- 时间限制用来设置路由的生效时间，两个各自都可为空
- 超时时间用来配置连接和响应的超时时间
- 限制只有某些ip可以通过路由可以设置允许ip,通过 ',' 分隔多个ip
- 请求类型即是请求方法，用来设置可以接受的请求方法,',' 号分隔, 如: GET,POST
- 最大请求速率和令牌桶容量用来配置限流的两个参数
- 挡板是用来阻止访问后端服务的，这里可以配置挡板是否启用
