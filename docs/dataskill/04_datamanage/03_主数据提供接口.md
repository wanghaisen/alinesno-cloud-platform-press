# 主数据提供接口

## 概述

在组织内部制定了标准规范，主数据管理平台按标准规范生成主数据后，向外提供共享接口。业务系统根据接口获取主数据信息，满足业务系统的使用需求

## 本内容你将获得

- 主数据管理对外提供的接口清单
- 主数据管理对外提供的接口服务

## 接口清单

| 接口服务名                | 功能说明                  |
| ------------------------- | ------------------------- |
| getStandardList           | 标准规范清单              |
| getStandardByID           | 标准规范明细              |
| getCatagoryList           | 数据目录清单              |
| getCatagoryByID           | 数据目录明细              |
| getDataDetailByStandardID | 主数据_通过标准规范id查询 |
| getDataDetailByCataID     | 主数据_通过数据目录id查询 |
| getDataDetailByName       | 主数据_通过主数据名称查询 |

## 接口服务

### 标准规范清单

| 请求地址 | http://xxx/api/IF/DataPublish/getStandardList |
| -------- | --------------------------------------------- |
| 请求方式 | get                                           |

**返回数据说明**

| 名称            | 类型   | 备注       |
| --------------- | ------ | ---------- |
| id              | string | 标准id     |
| standardNo      | string | 标准号     |
| standardName    | string | 标准名称   |
| releaseVersion  | string | 标准版本号 |
| issueDate       | string | 发布日期   |
| implementDate   | string | 实施日期   |
| competentDept   | string | 主管部门   |
| centralizedDept | string | 归口部门   |
| publishDept     | string | 发布单位   |
| remark          | string | 备注       |

例子：http://xxx/api/IF/DataPublish/getStandardList
