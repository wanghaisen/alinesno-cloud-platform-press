# 中台公开课

## 概述

公开课程的目标，为了更好的学习和落地中台，有一定的教学输出

- 成长输出
- 中台输出
- 方案输出

## 课程内容

这里体现从 0 到 1 的成长路线，对照的的中台能力模型[打开](../group/02_%E7%AE%A1%E7%90%86%E4%BD%93%E7%B3%BB/19_%E8%83%BD%E5%8A%9B%E6%A8%A1%E5%9E%8B.md)进行的成长和培养规划

| 序号 | 层级       | 目标                                                                        | 备注 |
| :--: | ---------- | --------------------------------------------------------------------------- | ---- |
